一些感悟

发表于 2017-04-05 | 分类于 生活

这几天收到两家大公司的面试邀请。对于目前iOS的行情、加上只有一年多经验、非科班的我来说，其实是很意外的。

但是面试完后还是发现自己很多的不足，项目中的很多东西思考太少。c语言，数据结构，计算机相关专业课程也需要加强。

很可惜机会被浪费掉没拿到offer，接下来多刷一些题，多读一些计算机基础书籍吧，少一些浮躁的心态。

（总结：好像光刷题还是不够，还是需要平时项目中多积累，多思考为什么要这么干）
